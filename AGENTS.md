# OMG Swarm Agents

When orchestrating a team, you must select the correct capability tier for the agent.

## Available Agent Profiles

| Type | Tier | Cost | Recommended Scenario |
|---|---|---|---|
| `writer` | Flash | Low | Writing Readme docs, extracting regex from strings. |
| `qa-tester` | Pro | Med | Running unit test suites and debugging the output stack traces. |
| `executor` | Pro | Med | General purpose software engineering (writing features). |
| `planner` | Ultra | High | Generating the `implementation_plan.md` artifacts. |
| `architect` | Ultra | High | Reviewing the system design before massive codebase shifts. |

*Remember to instruct the user to use `team <N>:<Tier>` to initialize these agents.*
